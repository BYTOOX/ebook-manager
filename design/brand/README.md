# Aurelia Brand Assets

- `source/` keeps the original logo files provided for the project.
- Runtime web assets are exported in `frontend/public/brand/`.
- Android launcher icons are generated into `android-reader/app/src/main/res/mipmap-*`.

Usage:

- Full horizontal logo: wide app chrome and headers.
- Small stacked logo: boot and authentication screens.
- Icon mark: favicon, install icons, compact empty states, and Android launcher icons.
