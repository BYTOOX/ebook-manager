# Prompt Codex - Phase 1

Tu vas commencer la Phase 1 du projet Aurelia.

Important :
- Ne développe pas toute l’application.
- Ne pars pas hors scope.
- Ne crée pas de fonctionnalités V2.
- Commence par inspecter le repo.
- Si le repo est vide, propose puis crée une structure propre.

Objectif Phase 1 :
Créer le socle backend et frontend minimal, prêt à évoluer.

Livrables :
1. Structure du repo.
2. Backend FastAPI minimal.
3. PostgreSQL configuré.
4. Alembic configuré.
5. Modèles DB initiaux.
6. Auth mono-utilisateur basique.
7. Frontend minimal avec thème dark/gold.
8. Podman Compose.
9. README de démarrage.

Stack :
- Backend FastAPI
- PostgreSQL
- SQLAlchemy 2.x ou SQLModel
- Alembic
- Frontend React/Next.js TypeScript
- Podman Compose

Avant de coder :
- Résume l’architecture que tu vas créer.
- Liste les fichiers que tu vas modifier/créer.
- Attends validation.

Après validation :
- Implémente étape par étape.
- Donne les commandes pour lancer.
