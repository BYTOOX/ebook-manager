# Prompt Codex - Phase 3 UI Premium

Tu vas implémenter l’interface mobile-first premium d’Aurelia.

Objectif :
Créer l’expérience visuelle principale :
- Home
- Library
- Book Detail
- Settings minimal

Direction artistique :
- Netflix-style browsing
- Apple-like sobriety
- dark cyber luxury
- noir profond
- accents jaune-or
- interface mobile Android prioritaire
- premium, sobre, élégante

Ne pas créer :
- dashboard admin générique
- interface desktop d’abord
- design Bootstrap standard
- couleurs trop vives

Écrans à implémenter :
- Home avec Continue Reading, Downloaded, Recently Added, Bottom navigation.
- Library avec Header, Search, Grid/List toggle, Filter chips, Sort, Grid de livres.
- Book Detail avec cover, titre, auteurs, rating, status, progress, Lire, offline placeholder, metadata, menu avancé.
- Settings minimal avec Appearance, Reading, Offline, Advanced.

Contraintes :
- responsive mobile-first
- composants propres
- CSS variables pour thème
- thème black/gold obligatoire
- mode automatique selon système prévu
- navigation bottom visible sauf reader

Avant de coder :
- Présente la structure de composants.
- Présente les variables de thème.
- Attends validation.
